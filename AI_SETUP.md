# Configuração da IA para Geração de Landing Pages

## Visão Geral

O editor de landing pages possui integração com IA para gerar conteúdo automaticamente. A IA está integrada via backend e funciona de forma **100% gratuita**, usando análise inteligente de prompts e templates otimizados para conversão.

## Como Funciona

A IA funciona de forma **completamente gratuita** através do backend. O sistema:

1. **Analisa o prompt** do usuário para entender o tom e foco desejado
2. **Identifica palavras-chave** como "emocional", "técnico", "urgente", "resultados práticos"
3. **Gera conteúdo personalizado** baseado em templates otimizados para conversão
4. **Adapta o conteúdo** ao título e descrição do curso selecionado

### Análise Inteligente de Prompts

O sistema detecta automaticamente:
- **Tom emocional**: quando o prompt menciona "transformação", "mudar vida", etc.
- **Tom técnico**: quando menciona "profissional", "certificado", etc.
- **Urgência**: quando menciona "limitado", "oferta", "urgente", etc.
- **Foco em resultados**: quando menciona "prático", "resultados", etc.

### Sem Necessidade de Configuração

✅ **Não é necessário configurar API keys**  
✅ **Funciona imediatamente após o deploy**  
✅ **100% gratuito e sem limites**  
✅ **Gera conteúdo otimizado para conversão**

## Como Usar

1. Acesse o editor de landing pages: `/creator/pages/new` ou `/creator/pages/edit/:id`

2. Selecione um curso (recomendado para melhor resultado)

3. Na seção "Gerador de Conteúdo com IA", digite um prompt descrevendo o que você quer na landing page

4. Clique em "Gerar com IA"

5. Visualize o resultado na área de pré-visualização

6. Clique em "Aplicar ao Editor" para usar o conteúdo gerado

## Exemplos de Prompts

- "Landing page persuasiva para curso de React, focada em resultados práticos e transformação de carreira"
- "Crie uma página de vendas emocional que mostre como o curso pode mudar a vida do aluno"
- "Landing page profissional com foco em benefícios técnicos e certificação"
- "Página de vendas com urgência e escassez, destacando oferta limitada"

## Notas

- O sistema funciona melhor quando um curso está selecionado, pois usa as informações do curso para gerar conteúdo mais relevante
- O conteúdo gerado pode ser editado manualmente após ser aplicado
- A visualização em tempo real mostra tanto o conteúdo do editor quanto o resultado da IA
